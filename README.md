<h3>Starting this repository:</h3>

```bash
# Clone this repository
$ git clone https://github.com/iwataniKenji/event-platform

# Access the project directory
$ cd event-platform

# Install dependencies
$ yarn

# Initialize and open local host
$ yarn start
```
